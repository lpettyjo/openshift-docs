= Dynamic provisioning
include::_attributes/common-attributes.adoc[]
:context: dynamic-provisioning

toc::[]

[role="_abstract"]
In dynamic provisioning, instead of a manually creating a pool of persistent volumes (PVs), an administrator creates a storage class. Using the storage class, {product-title} automatically triggers the storage backend to create a brand-new volume of the exact size and type requested, creates the PV object, and then the PV binds to the persistent volume claim (PVC).
